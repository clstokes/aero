# aero

`aero` provides instance metadata in a cloud-agnostic manner.

## Usage

```
$ aero which
amazon

$ aero list
address-private
address-public
instance
region
zone

$ aero read address-private
10.0.0.12

$ aero read zone
us-east-1a
```
